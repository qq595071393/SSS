功能说明 - 图标作用与单词点击功能

核心功能

智能字幕同步、单词点击释义、语音朗读、循环练习、循环计数

播放控制图标

  • 播放/暂停 点击开始播放视频，再次点击暂停播放
  • 上一句 跳转到上一句字幕的时间点
  • 下一句 跳转到下一句字幕的时间点
  • 循环播放 开启/关闭循环播放模式，重复练习当前片段
  • 循环计数 显示当前句子的循环播放次数，帮助跟踪练习进度
  • 字幕模式 切换字幕显示模式（中英/仅英/仅中/隐藏）

循环计数功能

智能跟踪每次循环练习的次数，帮助您更好地掌握学习进度

循环计数功能特点：

  • 自动计数 每次循环播放时自动增加计数，实时显示练习次数
  • 进度跟踪 通过循环次数了解自己对当前句子的掌握程度
  • 智能重置 切换到新句子时自动重置计数，开始新的练习
  • 可视化显示 大字体显示循环次数，清晰易读，便于快速查看

速度控制图标

  • 减速 降低播放速度，便于跟读练习
  • 加速 提高播放速度，快速浏览内容
  • 重置速度 将播放速度重置为1.0x正常速度

单词点击功能

点击任何英文单词即可查看详细释义和发音

点击下面的单词体验功能：

Hello world this is amazing tool

单词点击功能特点：

  • 智能识别 自动识别英文单词，支持连字符和缩写
  • 详细释义 显示中文释义、音标、词性、例句等信息
  • 语音朗读 美式英语发音，清晰准确，可调节朗读速度
  • 丰富词库 包含大量英语词汇，覆盖常用单词和短语

文件上传图标

  • 上传视频 选择英语学习视频文件（支持MP4、AVI、MOV等格式）
  • 上传字幕 选择对应的SRT字幕文件，系统自动同步中英文字幕
